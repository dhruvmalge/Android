Download and extract these files

in Android Studio copy and run this code and run in this main.dart
just by importing the file

-- import server_comm_http.dart

and add it in runApp().

The python file should be run on terminal using flask enabling server.
Just change the directory where the file is saved and run using 
--python <python_file>.py

and your server started.

Put the server obtained IP address in the webbrowser and also in dart file.
you will see it as the http://192.168.xx.xx:<port>

Run it and enjoy.

When the app will only display the streaming values if the server is started, wlse it will show spinning loading widget.
